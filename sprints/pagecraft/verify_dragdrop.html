How It Works:

  • Event delegation on #section-list handles dragstart, dragover, drop, dragend
  • dragstart: captures dragged element and index, adds .dragging class
  • dragover: calculates drop position (before/after midpoint), shows indicator
  • drop: calculates new index, reorders AppState.sections array, re-renders both views
  • dragend: cleanup (remove .dragging class and indicator)
