# PageCraft — Drag-and-Drop Landing Page Builder

## The Problem

Small business owners and freelancers need landing pages for campaigns, product launches, and lead capture — but existing builders (Wix, Squarespace) are overkill, slow, and subscription-heavy. They just want to pick a template, customize the content, preview it, and export clean HTML they can host anywhere.

## The Vision

**PageCraft** is a browser-based landing page builder that lets users create professional landing pages in minutes. Users pick a template, drag sections to reorder them, edit text and colors inline, preview on desktop and mobile, and export a standalone HTML file.

## What Users Can Do

1. **Choose a template** — Pick from 3 starter templates (SaaS product, event/webinar, portfolio showcase) displayed as visual cards with live previews
2. **Customize sections** — Drag-and-drop to reorder hero, features, testimonials, pricing, and CTA sections. Each section has inline-editable text and configurable accent color
3. **Live preview** — Toggle between desktop and mobile viewport previews. Changes reflect instantly — no save button, no reload
4. **Export HTML** — One click exports a self-contained HTML file with inlined CSS. The exported page looks identical to the preview and works offline

## Success Looks Like

A user opens PageCraft, picks the "SaaS Product" template, drags the CTA section above testimonials, changes the headline to "Ship Faster with PageCraft", picks a blue accent color, previews on mobile, and exports. The downloaded HTML file opens in any browser and looks exactly like the preview.
