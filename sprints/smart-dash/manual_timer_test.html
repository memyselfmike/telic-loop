Pomodoro Timer Manual Verification

Test 1: Initial State

Open the dashboard

Expected: Timer shows "25:00", mode shows "WORK", completed count shows "0"

Test 2: Start Button

Click Start button, wait 3-5 seconds

Expected: Display counts down (25:00 → 24:59 → 24:58...)

Test 3: Pause Button

Click Pause button while timer is running

Expected: Countdown stops at current time

Test 4: Resume After Pause

Click Start again after pausing

Expected: Countdown resumes from where it stopped

Test 5: Reset Button

Click Reset button (while running or paused)

Expected: Timer returns to "25:00" and stops

Test 6: Timer Completion (Accelerated)

Manual code edit required: Temporarily change line 480 from 25 * 60 to 3 (3 seconds)

Reload page, click Start, wait for timer to reach 00:00

Expected:

  • ✓ Beep sound plays
  • ✓ Mode switches to "BREAK"
  • ✓ Display shows "05:00"
  • ✓ Completed pomodoros: 1
  • ✓ Timer automatically starts counting down (04:59...)

Test 7: BREAK Mode Completion

Continue from Test 6 - let the BREAK timer complete

Expected:

  • ✓ Beep plays
  • ✓ Mode switches back to "WORK"
  • ✓ Display shows "00:03" (or 25:00 if using normal duration)
  • ✓ Completed pomodoros: 1 (unchanged - only increments on WORK completion)
  • ✓ Timer auto-starts
